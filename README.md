Java programlama dili ile JavaFX GUI kullanarak basit bir metin referans takip uygulaması yaptım.
HTML metin editörü içinde köşeli parantez içine yazılan sayıları kullanara seçili kütüphanedeki kitabın bilgilerini kullanarak kaynakça oluşturur.
CSV dosya işlemlerini kullanabilmek için basit bir kullanıcı giriş ve kullanıcı kayıt ekranı kullanıldı.

Programı kullanabilmek için kendi hesabınızı oluşturup her kullanıcıya "default" olarak atanan kütüphaneyi referans vermeyi deneyebilirsiniz.
